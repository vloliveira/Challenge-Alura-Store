# 📊 Análise de Desempenho de Lojas - Projeto de Decisão Estratégica

**Objetivo:**  
Ajudar o empresário João a identificar qual de suas 4 lojas deve ser vendida, com base em dados reais de desempenho.

## 🔍 O Que Analisamos
- 💰 Faturamento por loja
- 🛒 Categorias de produtos mais/menos vendidos
- ⭐ Avaliação dos clientes
- 🚚 Custos de frete
- 📈 Desempenho por produto

## 📌 Principais Conclusões
- Loja 3 se destacou como a melhor em faturamento e avaliação
- Loja 4 apresentou os piores resultados em todos os indicadores
- Recomendação clara: vender a Loja 4

## 🛠️ Como Usar
1. Baixe os arquivos do projeto
2. Abra no Google Colab ou Jupyter Notebook
3. Execute as células para ver:
   - Gráficos comparativos
   - Análises detalhadas
   - Recomendações estratégicas

## 💡 Insights Valiosos
- Como identificar os problemas de cada loja
- Quais métricas realmente importam
- Como tomar decisões baseadas em dados

## 👨‍💻 Tecnologias Usadas
- Python (Pandas, Matplotlib)
- Google Colab
- Análise de dados exploratória
